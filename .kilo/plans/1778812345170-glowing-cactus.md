## Proposed Fix

### 1. Rebuild Lesson Plan JSON — 6 Stages

The Shape E template expects 3 stages, but this lesson has 6 distinct pedagogical phases. Split cleanly:

| Stage | Name | Content | Time |
|-------|------|---------|------|
| 1 | Lead-in | Phubbing image discussion, bridge to politeness topic | 8 min |
| 2 | Vocabulary & Pre-reading | Exercise A1 (vocab fill-in) + Exercise B (rating survey) | 9 min |
| 3 | Video listening | Table Manners video + While you watch Q1-Q3 | 8 min |
| 4 | Reading for gist | Exercise C1 (topic sentences) + Exercise C2 (match main ideas) | 7 min |
| 5 | Reading for detail | Exercise D (close reading — 7 blanks) | 6 min |
| 6 | Discussion & reflection | Exercise E (critical thinking) + reflection on opening question | 8 min |

Total: 46 min. Each stage gets its own clean stage entry in the JSON with `stage_number`, `stage`, `stage_aim`, etc.

### 3. Design Rules Enforced

| Rule | Implementation |
|------|---------------|
| **Timer pills** | ONLY on task slides (A1, B, Video, C1, C2, D, E). NEVER on answer or transition slides. |
| **Answer slides** | green bg `#1e7e34` · table with columns · fragment `answer-correct` clickthrough · no timer |
| **Transition slides** | red bg `#c0392b` · fa-forward icon · directive + foreshadow + engagement · no timer |
| **Fragment policy** | ONLY used for answer reveals (answer-correct/answer-incorrect on green slides) |
| **B1 audience** | direct "you" imperatives · ≤15 word sentences · no passive voice · no banned B2+ vocabulary |
| **Exercise labels** | A1, B, C1, C2, D, E precisely labeled on every task and answer slide heading |
| **Logo** | `assets/logo.png` from `templates/ACT.png` · max-height 180px, max-width 60% |
| **B (survey)** | Task slide only — no answer slide (subjective ratings, no right/wrong) |
| **C1** | Task slide only — no answer slide (teacher checks by discussion) |

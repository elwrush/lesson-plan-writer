### Approach

Use Pillow for image processing. Strategy:
1. **Download from Pixabay** using the `large` image URL (typically 1280px wide) — already a reasonable size
2. **Resize** to max 1920px width (standard 1080p presentation resolution) while maintaining aspect ratio
3. **Compress** as JPEG with quality=85 (near-lossless visually, ~60-80% smaller than PNG)
4. **Strip EXIF/metadata** to save ~50-200KB per image
5. **Cache** images by Pixabay image ID in `output/.image-cache/` to avoid re-downloading the same image across lessons
6. **Target file size**: ~150-300KB per image (vs 2-8MB for original Pixabay downloads)

### Storage Structure

```

  .image-cache/           # Global image cache
    <pixabay_id>.jpg      # Optimized image, reused across lessons
  <subfolder>/
    slides/
      <date>-<topic>-slides.md
```

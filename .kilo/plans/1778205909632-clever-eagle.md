# Plan: Move pixabay-image-search Skill to Global Location

## Situation

The `pixabay-image-search` skill was created in the project at `.kilocode/skills/pixabay-image-search/` but needs to be global (user-level) so it works across all projects.

## Background

- **Global skills** go in `~/.kilo/skills/` (Windows: `C:\Users\<user>\.kilo\skills\`)
- **Project skills** go in `.kilo/skills/` within each project
- The skill's `SKILL.md` should be at the global location
- The script `scripts/pixabay_download.py` stays in the project

## Implementation

### Step 1: Verify global skills directory exists
`C:\Users\elwru\.kilo\skills\`

### Step 2: Move the skill

1. Create directory `C:\Users\elwru\.kilo\skills\pixabay-image-search\`
2. Copy `.kilocode/skills/pixabay-image-search/SKILL.md` → `C:\Users\elwru\.kilo\skills\pixabay-image-search\SKILL.md`
3. Update the SKILL.md to reference the script by absolute path: `C:\PROJECTS\LESSON PLAN WRITER 3\scripts\pixabay_download.py`
4. Delete the local copy at `.kilocode/skills/pixabay-image-search/`

### Step 3: Update SKILL.md script path reference

Change from relative reference to absolute path since the skill is global but the script is project-local.

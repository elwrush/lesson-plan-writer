## Then: Add Pedagogical Guided-Solve Slides

## Goal

Insert pedagogical slides after each transition slide that provide a **guided solve** — teaching students *how* to approach the upcoming question type by working through the first question as a class.

## reveal.js Research Results

### Fragments (built-in, no plugin needed)

Fragments are controlled via CSS classes. Available built-in classes:

| Class | Effect |
|---|---|
| `.fragment` (default) | Fade in (opacity 0→1) |
| `.fragment fade-up` | Slide up 40px + fade in |
| `.fragment fade-down` | Slide down 40px + fade in |
| `.fragment fade-left` | Slide left 40px + fade in |
| `.fragment fade-right` | Slide right 40px + fade in |
| `.fragment highlight-red` | Turn red when visible |
| `.fragment highlight-green` | Turn green when visible |
| `.fragment highlight-blue` | Turn blue when visible |
| `.fragment strike` | Add strikethrough |

Usage in markdown (via `<!-- .element: -->`):
```markdown
First point <!-- .element: class="fragment fade-up" -->

Second point <!-- .element: class="fragment" data-fragment-index="2" -->
```

Default order is document order. `data-fragment-index` overrides order.

### Auto-Animate (built-in, on by default)

- Add `data-auto-animate` to consecutive `<section>` elements
- Matches elements by content/text or `data-id` attributes
- Animates position, size, color, font-size, etc. between matching elements
- Unmatched elements fade in via `data-auto-animate-unmatched="fade"` (default behavior)
- Config options: `autoAnimateDuration: 1.0`, `autoAnimateEasing: 'ease'`

In markdown slides, applied via slide directive:
```markdown
<!-- .slide: data-auto-animate data-background="#1a6b5a" -->
## Question

This is the question
```

### Key Insight for Pedagogy Slides

For revealing step-by-step *thinking*, fragments are more predictable and easier to control than auto-animate. Auto-animate is better for cinematic transitions between slides with matching content.

**Recommendation**: Use **fragments** to progressively reveal the thinking process (question → hint → elimination steps → answer) within each pedagogical slide. Use **auto-animate** only for the transition *between* the transition slide and the pedagogical slide (to smoothly carry the heading forward).

## Three Pedagogical Slides to Create

### 1. T/F Guided Solve (after transition-3, before task-3)

**Background**: `#1a6b5a` (teal)
**Purpose**: Teach the True/False strategy using Exercise 2, Q1 as a worked example

Step-by-step reveal via fragments:
1. **Show the question**: "The author wrote the text to explore the generation gap and problems it can cause, and to suggest a possible solution." + the article paragraphs (condensed to key lines)
2. **Hint fragment**: "Look for keywords in the statement, then find them in the text. Does the meaning match?"
3. **Show evidence**: Highlight relevant sentence from paragraph A: "One of the most important jobs parents have is to teach their children about the world..." (NOT this one). Then paragraph F: "if we start to think of it as a source of learning and growth..."
4. **Answer fragment**: "✓ **TRUE**" with highlight-green
5. **Explanation**: "The text does explore the gap and suggest a solution."

### 2. (Optional) Paragraph Matching Guided Solve

**Background**: `#1a6b5a` (teal)
**Purpose**: Teach the matching strategy using Exercise 3, Q1

1. **Show statement**: "The generation gap affects people in their job because people of different ages working together may have very different opinions."
2. **Hint**: "Look for the paragraph that talks about the generation gap at work."
3. **Show paragraphs B vs A**: Compare — A defines the gap in families, B talks about it at work.
4. **Answer**: "✓ **B**"

### 3. Multiple Choice Guided Solve (after transition-4, before task-4)

**Background**: `#1a6b5a` (teal)
**Purpose**: Teach elimination strategy using Exercise 4

1. **Show question**: "Which statement gives the best conclusion of the article?"
2. **Show all 3 options** (a, b, c)
3. **Eliminate a**: "Learning and change are both easier for younger people" — ✗ This is mentioned but it is not the *main* conclusion. The article is about bridging the gap, not just saying young people learn faster.
4. **Eliminate b**: "Technology often causes conflict between young and old" — ✗ Too narrow. Technology is only one small example in paragraph D.
5. **Answer c**: "Differences can be good if we take time to understand them" — ✓ This matches the main message of the article.

## Script

The script uses `pymupdf` (fitz) to manipulate PDFs. It creates a new PDF with:
- Page 1: A narrow header (school logo + "Anglo Singapore International School" text) at the top
- Pages 2+: Full-page inserts of the requested source PDF pages

### Python Implementation

```python
import fitz  # pymupdf
import sys
from pathlib import Path

HEADER_HEIGHT = 60  # points — narrow header band

def create_header_page(doc, logo_path, text):
    """Create a blank A4 page with a narrow header band."""
    page = doc.new_page(width=595, height=842)
    # White background
    page.draw_rect(fitz.Rect(0, 0, 595, 842), color=None, fill=(1, 1, 1))
    # Header band background (dark blue)
    page.draw_rect(fitz.Rect(0, 0, 595, HEADER_HEIGHT), color=None, fill=(0.1, 0.14, 0.21))
    # Logo (small, left-aligned in header)
    logo_rect = fitz.Rect(15, 8, 75, HEADER_HEIGHT - 8)
    page.insert_image(logo_rect, filename=logo_path, keep_proportion=True)
    # School name text (to right of logo)
    page.insert_text(
        fitz.Point(85, HEADER_HEIGHT / 2 + 5),
        text,
        fontname="helv",
        fontsize=12,
        color=(1, 1, 1),
    )
    return page

def main(source_pdf, pages_str, output_pdf, logo_path, header_text):
    doc_out = fitz.open()
    
    # Page 1: header only
    create_header_page(doc_out, logo_path, header_text)
    
    # Parse page range
    pages = []
    for part in pages_str.split(","):
        part = part.strip()
        if "-" in part:
            start, end = part.split("-")
            pages.extend(range(int(start), int(end) + 1))
        else:
            pages.append(int(part))
    
    # Insert source pages
    doc_src = fitz.open(source_pdf)
    for p in pages:
        # p is 1-based, fitz uses 0-based
        src_page = doc_src[p - 1]
        # Create full-page insert (no header)
        page_out = doc_out.new_page(width=595, height=842)
        # Show the source page, scaled to fit the full A4 page
        page_out.show_pdf_page(
            page_out.rect,
            doc_src,
            p - 1,
        )
    
    doc_out.save(output_pdf)
    doc_out.close()
    doc_src.close()
    print(f"Saved to {output_pdf}")

if __name__ == '__main__':
    main(
        source_pdf=sys.argv[1],
        pages_str=sys.argv[2],
        output_pdf=sys.argv[3],
        logo_path=r'C:\PROJECTS\LESSON-PLAN-WRITER-3\templates\ACT.png',
        header_text='Anglo Singapore International School',
    )
```

### 3. Implementation Notes

- **`pymupdf`** is already available (used by `pdf_to_pptx.py` and other scripts)
- **Logo**: Use `templates/ACT.png` (1396x1732, portrait) — scaled proportionally within the header band
- **Output path**: Default to `output/{source_pdf_stem}_with_header.pdf` or ask the user
- **Header dimensions**: 60pt tall (narrow), full width. Dark navy background (#1a1a2e ≈ 0.1, 0.14, 0.21 RGB)
- **Page 1**: Header only, rest of page blank (or could add a title/cover text if desired)
- **Pages 2+**: Source pages inserted full-page using `show_pdf_page()` which preserves all original content
- **Page numbering**: User input is 1-based (human-readable). Convert to 0-based for fitz.

### 4. Edge Cases

- **Invalid page numbers**: Check against source PDF page count, report error if out of range
- **PDF not found**: Check file exists before processing
- **Logo not found**: Fall back to text-only header
- **Empty page range**: Error — must specify at least one page

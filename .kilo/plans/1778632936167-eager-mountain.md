## Overview
Create a new skill at `.kilo/skills/typst-document-creator/SKILL.md` — entirely separate from the existing `typst-author` skill. The new skill is linked to the existing `/write-typst-document` command.

## Files to Create

### 1. `.kilo/skills/typst-document-creator/SKILL.md` — New skill

**Skill metadata:**
- name: `typst-document-creator`
- description: Creates new Typst documents from scratch or transforms existing lesson materials. Manages the full pipeline: user consultation → code generation → PDF compilation → integrity verification.

**Workflow steps (as `SKILL.md` content):**

#### Step 0: Prerequisites
- Load `typst-author` skill for syntax reference
- Verify `typst compile` is available (skip with warning if missing)
- Check for `typstyle` (optional, skip formatting if missing)

#### Step 1: Source Consultation
Ask user:
- **Source**: "Create from scratch" or "Transform existing lesson materials"?
  - If "Transform existing": ask for file path(s) — JSON, markdown, text, or other
  - If "Create from scratch": ask for the document purpose/topic

#### Step 2: Header Selection
Ask user:
- **Header style**: "Official school header" or "Custom/None"?
  - **Official header layout** (left-ear/right-ear arrangement with rule line):
    - Left ear: `templates/ACT.png` (image)
    - Right ear: `templates/1135082720.png` (image)
    - Title centered: **"Mathayom Program"**
    - Rule line underneath (full width)
    - This exact layout should be a reusable template file at `templates/mathayom-header.typ`
  - **Custom/None**: Ask what they want instead, or skip header

#### Step 3: Print Format Consultation
Ask user:
- **Print format**: "Single A4 sheet" or "Booklet (A5 pages, folded)"?
  - **A4 (default)**: 11pt body font
  - **Booklet (A5)**: Research font size via web search. Typical guidance: A5 booklets printed 2-up on A4 get scaled ~70%. Body text ~14pt (so scaled A5 reads ~10pt effective). Use Tavily search to confirm best practice. Present finding to user before proceeding.

#### Step 4: Generate Typst Code
- Consult repomix-packed Typst codebase at `knowledge-base/typst-packed.json` for correct syntax (never hallucinate)
- Create output at `typst/code/{subfolder}/{filename}.typ` where subfolder names match document purpose
- Include:
  - `#import "templates/mathayom-header.typ": header` if official header selected
  - Page setup (A4 portrait or A5 booklet page dimensions)
  - Font size per Step 3 decision
  - Content from either transformed materials or scratch authorship

#### Step 5: Compile PDF
- `typst compile "typst/code/{subfolder}/{filename}.typ" "typst/PDF/{subfolder}/{filename}.pdf"`
- Echo the output subfolder path in `typst/PDF/` for user reference

#### Step 6: Formatting
- `typstyle --check` on the generated `.typ` file
- Apply `typstyle -i` only if changes are limited to newly created code

#### Step 7: Integrity Check (Deterministic)
- **Purpose**: Ensure transferred text from existing materials has NOT been truncated, edited, hallucinated, or changed
- **Method**: 
  - For each source file transformed, read the original content
  - Extract all content strings from the generated `.typ` file (between `[...]` content blocks, string literals)
  - Compare source text vs. Typst text using character-by-character diff (`difflib.SequenceMatcher` in Python)
  - Report: "INTEGRITY PASSED: 0 differences found" or "INTEGRITY WARNING: N differences found" with specific locations
  - Halt if differences exceed a threshold (e.g., >1% of characters differ)

#### Step 8: Report
- Output file paths (`.typ` source + `.pdf`)
- Formatting applied (yes/no)
- Compilation result (success/error + line number)
- Integrity check result (passed/warning)

### 2. `templates/mathayom-header.typ`

A reusable Typst module providing the official school header:

```typst
// mathayom-header.typ — Official Mathayom Program header
// Usage: #import "templates/mathayom-header.typ": mathayom-header
#let mathayom-header() = {
  block(
    stroke: (bottom: 0.5pt + black),
    inset: (bottom: 6pt),
    grid(
      columns: (1fr, 2fr, 1fr),
      align: (left + horizon, center + horizon, right + horizon),
      image("templates/ACT.png", height: 1.2cm),
      text(size: 16pt, weight: "bold")[Mathayom Program],
      image("templates/1135082720.png", height: 1.6cm),
    ),
  )
}
```
